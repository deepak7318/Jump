Jump
====

Sublime Text 3 motion plugin

![privet1mb](https://raw.github.com/england/Jump/master/privet1mb.gif)

### Manual installation

    git clone git@github.com:england/Jump.git ~/.config/sublime-text-3/Packages/Jump

### Default keymap

`ctrl+'`, `<character>` prepare jump

and now:

`<character>` jump

`escape` cancel jump
